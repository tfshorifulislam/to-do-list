# To-Do List

**To-Do List** is a simple and responsive web application that helps users manage their daily tasks efficiently. It allows users to add, organize, and track tasks with a clean and user-friendly interface.

## 🌟 Features
- ➕ Add new tasks easily
- ✅ Mark tasks as completed
- ❌ Delete tasks when finished
- 📱 Responsive design for different devices
- ⚡ Simple and fast user experience

## 💻 Technology Stack
- HTML5
- CSS3
- JavaScript (Vanilla JS)
